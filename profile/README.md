### What we build

**[Siza](https://github.com/Forge-Space/siza)** â€” The open full-stack AI workspace. Generate components, scaffold apps, and ship to production with any AI provider (BYOK). Built with Next.js, Supabase, and Cloudflare Workers.

**[Siza Desktop](https://github.com/Forge-Space/siza-desktop)** â€” Native desktop app with local AI generation via Ollama. Available for macOS, Windows, and Linux.

#### Generation Engine

**[siza-gen](https://github.com/Forge-Space/siza-gen)** â€” AI generation engine powering Siza. **[Siza MCP](https://github.com/Forge-Space/ui-mcp)** â€” MCP server for AI-powered UI generation. 21 tools, template packs, brand integration, and vector search. Connect to any MCP-compatible AI client.

### Get started

```bash
# Use Siza's generation engine
npm install @forgespace/siza-gen

# Or try the workspace
# https://forgespace.co

# Clone and run locally
git clone https://github.com/Forge-Space/siza.git
cd siza && npm install && npm run dev
```
